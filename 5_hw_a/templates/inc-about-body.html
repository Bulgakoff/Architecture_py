Москва
Основы Python Python входит в топ-10 самых востребованных языков программирования (по данным Stack Overflow). Он открывает путь в топовые IT-компании: Google, Pixar, Youtube, Instagram, Nasa, Intel, Pinterest используют именно его. После курса вы сможете устроиться в компанию, где создают интерактивные веб-сервисы на Python, а таких очень много: 3305 вакансий для python-разработчика открыто прямо сейчас на hh.ru.
Алгоритмы и структуры данных на Python Курс предназначен для тех, кто уже знаком с языком программирования Python, но хотел бы понять и изучить базовые структуры данных, научиться реализовывать алгоритмы, знать где и как их использовать и для чего они нужны.
Основы Django Вы занимаетесь программированием и интересуетесь веб-разработкой? Возможно, имеете определенный практический опыт разработки с использованием РНР и СУБД? На данном курсе Вы изучите создание веб-приложений на языке Python с использованием MVC-подхода на примере фреймворка Django.
Основы Flask Flask - это фреймворк для веб-приложений, написанный на Python. Он разработан Армином Ронахером , который возглавляет международную группу энтузиастов Python по имени Pocco. Flask основан на наборе инструментов Werkzeug WSGI и шаблонизаторе Jinja2. Оба проекта Pocco.
